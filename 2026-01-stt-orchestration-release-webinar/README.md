You can reproduce locally the `STT orchestration` demo presented during the webinar in three simple steps:


1. Clone the repository
   ```bash
   git clone https://github.com/pyannoteai/tutorials.git
   ```  
2. Move to this directory
   ```bash
   cd tutorials/2026-01-stt-orchestration-release-webinar
   ```
3. Run the notebook with [`uv`](https://docs.astral.sh/uv/)
   ```bash
   PYANNOTEAI_API_KEY=... uv run jupyter lab demo.ipynb
   ```
